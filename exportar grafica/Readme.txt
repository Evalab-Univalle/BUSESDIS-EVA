Pasos para la ejecucion del programa.

1. Instalar o tener instalada la version 1.8 de ruby

2. Instalar la gema o librería Igraph como se indica en este enlace http://igraph.rubyforge.org/igraph/

3. Desde una terminal, conceder permisos de ejecución para el archivo "main.rb". 

	chmod +x main2.rb

4. Ajustar los parametros en el archivo main2.rb, de acuerdo a los resultados obtenidos del algoritmo genetico.
	
	cromosoma
  	vecindad
  	cantidad_nodos
  	tipo_grafo
  	nodo_a
  	nodo_p
  	nodo_m

5. Los resultados para ilustrar el grafo quedará en el archivo "ver_grafo.graphml"

6. Las rutas de los nodos se encuentran en el archivo "rutas.txt"
